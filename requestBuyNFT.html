Request to buy NFT









Swaping NFTs on platform

As of security and trustability increment policies, the NFTs "change of registration" mechanism has been implemented in a uniquely brilliant way.

The person who wants to buy an NFT, submits a request containing an NFT token id which specifies the asset, the a seller address which specifies the current owner of the asset, and the price. The contract first checks if the NFT belongs to the seller, then freezes the price from the buyer's wallet. then if the seller accepts the request, NFT would be transferred to the buyer, and the frozen "MRY" would be transferred to the seller. otherwise, the frozen "MRY" would be transferred back to the buyer wallet.
